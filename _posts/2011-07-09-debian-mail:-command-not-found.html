 $aptitude search mailx 

p   bsd-mailx                                               - simple mail user agent                                           
p   heirloom-mailx                                          - feature-rich BSD mail(1)                                         
v   mailx   


$aptitude show mailx 
o current or candidate version found for mailx
Package: mailx
State: not a real package
Provided by: bsd-mailx, heirloom-mailx, mailutils


So i installed the mailutils package by :

#aptitude installl mailutils

now mail command can be found and issued.
